Lists

Lists are used to display items in an organized manner. There are different types of lists: ordered lists, unordered lists, and description lists.

Ordered List
A list where the items are numbered in sequence.
Unordered List
A list where the items are marked with bullets.
Description List
A list used to display key-value pairs or definitions.

Linking

Links allow users to navigate from one page to another. The most common HTML element used for linking is the <a> tag.

Formatting

HTML provides various tags for formatting text. These include tags for bold, italic, underlined, and other text styles.

Selectors

Selectors in CSS allow you to target specific HTML elements to apply styles.

Pseudo-Classes

Pseudo-classes allow you to target elements based on their state or position within the document.
